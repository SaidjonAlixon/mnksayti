## 1. PROJECT OVERVIEW

A corporate marketing and recruitment website for a U.S.-based trucking / logistics company. The site serves two business goals at once:

1. **Win clients (shippers / brokers)** — convince freight partners to trust us with their loads.
2. **Recruit drivers** — attract qualified CDL drivers to join the fleet.

The core creative idea is to **completely abandon the generic trucking template** (blue gradient hero + stock photo of a truck on a highway + three identical service cards + a testimonial slider) and instead fuse a **"Dispatch Terminal" aesthetic** with **premium editorial design**. The result feels engineered, confident, and expensive — like a control-room dashboard crossed with a high-end magazine.

## 4. DESIGN CONCEPT — "ASPHALT & SIGNAL"

This is the heart of what makes the site unique. The concept stands on four pillars.

### 4.1. "Dispatch Terminal" Layer
The interface borrows the language of a professional dispatch screen and a freight manifest:
- **Monospace data labels** for small text, numbers, and coordinates: `MC-1234567`, `48 STATES`, `LOAD #4821`, `ETA 06:42`.
- **Oversized display headlines** (condensed grotesque) that hit like newspaper front-page type.
- A **route line** — a thin signal-colored spine that runs through the entire site, connecting sections with "mile-marker" nodes. As the user scrolls, the line **fills in** like a live progress route. This is the signature element that ties every page together.

### 4.2. Editorial, Asymmetric Grid
No centered, three-column cookie-cutter layout. Instead:
- A magazine-style, **asymmetric grid** with intentional imbalance.
- Generous **negative space** paired with dense, high-contrast text blocks.
- **Numbers as design elements** — miles, years, and states are set huge and treated like typography, not footnotes.

### 4.3. Kinetic Motion
- Content slides **horizontally** on scroll — as if a truck just passed by.
- An **infinite marquee** ribbon in the hero scrolls phrases like `RELIABLE • ON-TIME • 48 STATES • FULLY INSURED •`.
- **Counter animations** — stats count up from zero when they enter the viewport.
- Subtle **grain / asphalt texture** overlay for an industrial, tactile feel.
- A **custom cursor accent** on desktop (small crosshair / target reticle) over interactive elements — reinforces the "terminal" feel.

### 4.4. Restraint
Dark mode, industrial, premium, "less is more." Signal-red appears **only** at the most important moments (primary CTAs, key emphasis), so it stays powerful instead of decorative.

### 4.5. Full-Scroll, Fully-Animated Experience (MANDATORY)
The site is **one continuous scrolling page**, not a set of separate page loads. This is a hard requirement.

**Single-page structure**
- All four sections — **HOME · ABOUT · DRIVERS · CONTACT** — live on one page as anchored sections (`#home`, `#about`, `#drivers`, `#contact`).
- The nav links **smooth-scroll** to their section; no full page reload ever happens.
- The active nav link (red underline) updates automatically based on which section is in view (scroll-spy).
- Smooth scrolling is enabled site-wide; an optional inertia/eased scroll can be layered on for a more premium glide.

**Animation everywhere (scroll-driven)**
Every section animates in response to scroll position. Required behaviors:

| Trigger | Animation |
|---|---|
| Page load | Hero sequence: headline lines rise in one after another, marquee starts, route line "ignites" |
| Section enters viewport | Scroll-reveal — content fades + translates up `24px`, staggered per element |
| Scroll progress | The **route-line spine fills** top-to-bottom in sync with scroll; mile-marker nodes pulse as they activate |
| Stats enter viewport | Numbers **count up from 0** (odometer feel) |
| Horizontal bands | Marquee ribbons and select rows **slide horizontally** as the user scrolls past |
| Hover (desktop) | Buttons, cards, and links have micro-interactions (slide, fill, lift) + reticle cursor |
| Section transitions | Backgrounds shift tone subtly between sections for depth |

**Quality floor (non-negotiable)**
- Animations are smooth (target 60fps), GPU-friendly (`transform` / `opacity` only).
- Fully respects `prefers-reduced-motion` — motion reduces to simple fades, the site stays usable.
- Scroll performance must not jank on mid-range mobile; heavy effects simplify on small screens.

## 5. DESIGN SYSTEM (TOKENS)

### 5.1. Color Palette — MNK Brand
Sampled directly from the **MNK Logistics LLC** logo (royal blue + signal red + white). The dark base is a deep **navy** (not pure black) so it reads as brand-blue territory, not generic black.

| Role | Color | HEX |
|---|---|---|
| Brand blue (primary) | MNK Royal Blue | `#0A4D9C` |
| Brand blue (bright accent) | Highway Blue | `#1E6FBF` |
| Brand red (signal / CTA) | MNK Red | `#E0141A` |
| Dark base (navy) | Deep Navy | `#08152E` |
| Dark surface | Navy Surface | `#0E2244` |
| Light base | White | `#FFFFFF` |
| Light surface | Off-white | `#F4F6FA` |
| Ink (text on light) | Navy Ink | `#0A1A33` |
| Text (on dark) | Off-white | `#EAF0F8` |
| Text (secondary) | Steel grey | `#8A97AC` |
| Border / divider | Hairline | `#1B335C` |

> **Usage rule:** Blue carries structure and trust (dark navy sections, links, headlines). Red is the *signal* — used only for primary CTAs, the route-line spine, and key emphasis. White/off-white sections keep the corporate, clean feel of the logo. Alternating **navy ⇄ white** sections give the full-scroll page its rhythm.

### 5.2. Typography Scale

| Use | Typeface (suggested) | Notes |
|---|---|---|
| Display / Headlines | Condensed grotesque (e.g. *Anton*, *Archivo Expanded*, *Bebas Neue*) | Bold, tight, uppercase for impact |
| Body | Neo-grotesque (e.g. *Inter*, *Geist*) | Clean, highly readable |
| Labels / Data | Monospace (e.g. *JetBrains Mono*, *Space Mono*) | For tags, numbers, coordinates |

Type scale (desktop): `H1 96px / H2 64px / H3 40px / H4 28px / Body 18px / Label 13px (mono, uppercase, tracked)`. Scales down responsively.

### 5.3. Spacing & Layout
- 8-point spacing system: `4 · 8 · 16 · 24 · 40 · 64 · 96 · 128`.
- Max content width: `1280px`; full-bleed sections allowed for hero/marquee.
- Asymmetric 12-column grid; sections intentionally break alignment for editorial tension.

### 5.4. Core Components
- **Buttons:** Primary (solid signal-red, square-ish corners), Secondary (outline), Ghost (text + arrow). Hover = slight slide + accent fill.
- **Cards:** Bordered, dark surface, mono label in the corner, large headline, hover lift.
- **Nav:** Sticky top bar (matching the supplied design), red underline on active link, mobile hamburger → full-screen overlay menu.
- **Forms:** Dark inputs, mono labels, inline validation, large touch targets.
- **Stat block:** Huge number + mono caption, count-up on scroll.
- **Route line:** Persistent vertical/horizontal spine with mile-marker nodes and scroll-fill.

## 9. MOTION & INTERACTION SPEC

- **Easing:** custom cubic-bezier `(0.16, 1, 0.3, 1)` for a confident, weighty feel.
- **Durations:** micro-interactions `150–250ms`; section reveals `600–800ms`.
- **Scroll reveals:** elements fade + translate up 24px as they enter the viewport.
- **Route line:** fills proportionally to scroll progress; mile-marker nodes "ping" (small pulse) as they activate.
- **Reduced motion:** all animation respects `prefers-reduced-motion` and degrades gracefully.
